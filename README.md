# 🗺️ ezGPX

## 🔎 Description
Easy to use Python GPX library.

## 🛠️ Installation

```bash
pip install ezgpx
```

## 🏁 Get started

```python
import ezgpx

test_gpx = ezgpx.GPX("file.gpx")
test_gpx.plot(start_stop=True, elevation_color=True)
```
